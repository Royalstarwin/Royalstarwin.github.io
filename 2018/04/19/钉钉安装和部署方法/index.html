钉钉安装和部署方法

发表于 2018-04-19

安装nvm

wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.33.5/install.sh | bash

执行完毕后，关闭当前终端，打开一个新终端，键入

nvm --version

0.33.5

证明nvm已成功安装

安装node

nvm install v8.9.0
node -v
v8.9.0

安装pm2

npm install pm2 -g
pm2 -v
2.7.1

端口

var port = normalizePort(process.env.PORT || '3000');
// 修改3000为任意端口，此端口需映射外网，并将外网地址填入钉钉官网微应用配置：首页地址
